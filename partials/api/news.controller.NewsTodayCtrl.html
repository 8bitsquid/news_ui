NewsTodayCtrl
controller in module news

Description

Convenience controller to be used with the newsCard directive, to display current news and events.

Dependencies

  • $scope
  • $filter
  • ualibNewsFactory

Example

<div ng-contorller="NewsTodayCtrl">
 <h2>News</h2>
 <div ng-repeat="item in news">
     <div news-card="item">
 </div>

 <h2>Events</h2>
 <div ng-repeat="item in event">
     <div news-card="item" news-type="event">
 </div>
</div>
