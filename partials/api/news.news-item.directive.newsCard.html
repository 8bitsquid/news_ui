newsCard
directive in module news.news-item

Description

Directive to render different types of news items in a condensed list. This is useful for lists of the most current items and not intended for rendering a list of the whole news archive

Usage

as attribute
<ANY news-card="{object}"
     [news-type="{string}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
newsCardobject

The news item JSON object from the API

newsType
(optional)
string

Used to load templates for different types of news items

Supported news types:

type template
news news-item/news-card.tpl.html
event news-item/news-card.tpl.html

(default: news)

Example

<h2>News</h2>
<div ng-repeat="item in news">
     <div news-card="item">
</div>

<h2>Events</h2>
<div ng-repeat="item in event">
     <div news-card="item" news-type="event">
</div>
